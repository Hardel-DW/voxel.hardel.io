# Composants de l'IDE pour Datapacks

Cette interface vous permet de créer des datapacks personnalisés de manière simple et intuitive. Voici les différents composants disponibles pour la conception :

- [ ] **Sélection d'entité**  
  Sélectionnez les entités disponibles dans le jeu pour interagir avec elles ou les manipuler dans vos datapacks.

- [ ] **Sélection de son**  
  Choisissez des sons parmi la liste des effets sonores du jeu pour créer des événements sonores ou des déclenchements personnalisés.

- [ ] **Sélection d'item**  
  Permet de choisir des objets parmi tous ceux présents dans le jeu pour les utiliser dans vos commandes ou événements.

- [ ] **Sélection de blocs**  
  Choisissez les blocs pour vos constructions, interactions ou autres fonctions dans les datapacks.

- [ ] **Sélection de particules**  
  Sélectionnez des effets de particules pour ajouter des visuels dynamiques et personnalisés à votre projet.

- [ ] **Sélection de dégâts**  
  Permet de définir et de sélectionner des types de dégâts (comme feu, chute, explosion) pour configurer des effets de dégâts précis.

- [ ] **Sélection d'événements de jeu**  
  Sélectionnez des événements de jeu tels que l'ouverture d'un coffre ou la chute d'un joueur pour déclencher des actions dans vos datapacks.

- [ ] **Sélection d'effet de potion**  
  Permet de choisir des effets de potion (comme force, régénération, invisibilité) pour les appliquer à des entités ou des joueurs.
