# Une Collaboration Majeure
Je suis ravis d'annoncer que le Configurateur d'Enchantements supporte désormais officiellement **Dungeons & Taverns**. C'est un datapack de structures avec plus de 3 millions de téléchargements sur Modrinth, qui est vraiment trés impressionnant et immersive.

## Qu'est-ce que le Configurateur d'Enchantements ?
Si tu ne connais pas Voxel, je te présente le Configurateur d'Enchantements, c'est un outil qui te permet de modifier les enchantements d'un mods ou d'un datapack, avec une interface possible simple pour tout public. Voici les fonctionnalités :
- Désactiver un enchantement
- Modifier les effets d'un enchantement
- Modifier les probabilités d'apparition ou l'équilibrage.
- Définir dans quelles structures l'enchantement peut être trouvé
- Modifier les objets qui peuvent supporter l'enchantement
- Modifier les compatibilités entre enchantements

## Et le rapport avec Dungeons & Taverns ?
Tu peux maintenant définir dans quelles structures de DnT, un enchantement peut être trouvé cela permet de personnaliser ton expérience de jeu.



# Une Expérience de Jeu Enrichie
L'intégration de Dungeons & Taverns ouvre de nouvelles possibilités, tu peux rendre un enchantement exclusif à certaines structures, ce qui permet de modifier la difficulté d'obtention d'un enchantement.

## Les objectifs Futurs
Notre vision reste la même, faciliter l'accès à une expérience de jeu personnalisable, et une meilleur compatibilités entre les contenus.
