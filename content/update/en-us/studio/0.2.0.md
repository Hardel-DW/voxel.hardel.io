## Features for the Version 0.2.0

- Added the management of "Primary" objects (items which can be enchanted) and
  "Supported" objects (This one defines which items can receive enchantment
  through Anvils, Structure rewards, trading...).
- A major bandwidth reduction with some performance improvements.
- The user interface has some fixes, including color adjustments.
- And few tiny fixes, especially when switching languages.
