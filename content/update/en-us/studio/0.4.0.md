## Features for the Version 0.4.0

The enchant's configurator has been rewritten. Maybe you can't see any
difference, it's a technical update :

- For the development team, now for adding a new features it takes less time.
- It will be now possible to create a different configurator for each Minecraft
  version.
- Also, it will be possible to create different configurators for other aspects
  of Minecraft.
- And it's will be possible to switch instantly from a configurator to another,
  without leaving the page.
- In the near future, a **configurator maker** will allow to **anyone** to
  create their own configurator.

#### Added Features

- It's now possible to disable an specific enchantment.
- A new debugging menu has been added.
- The "slot" tabs it's fixed and works perfectly.
