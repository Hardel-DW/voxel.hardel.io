## Features for the Version 1.1.0

Some new features, and some bugs have been fixed.

#### New Features

- **New button**: A new button named "Mode" in the "General" tab, it allows to
  define the mode of acquisition of enchantments.
  - **Default** : The enchantments can be obtained as they were planned by the
    creator, and of course according to your configuration.
  - **Delete** : For the related enchantments, all effects are removed, and the
    enchantment is no longer obtainable.
  - **Only Creative** : This mode, allows to only obtain the enchantments in
    creative mode.

- **Debug menu**: A new border color indicates when an element is deleted (Red),
  new (Green), modified (Gray) or unchanged (Gray).

#### Bug Fixes

- The **Compiler** now correctly deletes files, if it is empty.
