# Version 2.1.0 - Changements & Traductions

## Aperçu
Cette mise à jour se concentre sur trois domaines majeurs : une refonte complète de l'interface d'export/paramètres, une refonte massive du système de traduction et une toute nouvelle page pour visualiser toutes vos modifications.

# Refonte de l'Interface

### Export
Le bouton Export a été complètement repensé. Au lieu d'un petit popover, il ouvre maintenant une modale plein écran avec un design épuré et minimaliste.

![Modale d'Export](/images/patchnote/studio/2.1.0/export.webp)

La nouvelle interface d'export contient uniquement l'essentiel :
- **Sélection du format** - Choisissez votre format d'export
- **Boutons de soutien** - Moyens de soutenir le projet
- **Bouton de téléchargement** - Un bouton blanc surdimensionné, impossible à manquer

### Intégration GitHub
Toutes les fonctionnalités liées à GitHub ont été déplacées vers un bouton dédié entre Export et Paramètres. Cela inclut :
- **Pages dédiées** pour la gestion des dépôts
- **Design simplifié** - Plus propre et plus intuitif
- Fini les popovers encombrés

![Modale d'Export](/images/patchnote/studio/2.1.0/github.webp)

### Dialogue des Paramètres
Le bouton Paramètres ouvre maintenant un vrai dialogue au lieu d'un popover. Il contient actuellement la sélection de la langue, avec plus d'options à venir.

-# La page de débogage précédemment accessible dans les paramètres a été supprimée.

![Modale d'Export](/images/patchnote/studio/2.1.0/settings.webp)
---

# Refonte du Système de Traduction

Une refonte massive du système de traduction dans tout Voxel Studio.

### Pour les Utilisateurs
- **100% des textes sont maintenant traduits** - Sans exception, tout est localisé
- **Changement de langue instantané** - Changez de langue sans recharger la page
- **Aucune perte de données** - Vos modifications sont préservées lors du changement de langue

### Améliorations Techniques
- Toutes les clés de traduction sont maintenant chargées en une fois, éliminant l'approche de chargement divisé précédente
- Taille du bundle réduite et performances améliorées
- Les pages en dehors de Studio bénéficient du même système comme Harmonisation/Convertisseur, pages d'accueil et toutes les autres pages...

---

# Page des Changements

L'ajout le plus important de cette mise à jour. Vous pouvez maintenant visualiser exactement ce que Voxel Studio fait à vos packs.
Cette nouvelle page affiche :
- Quels fichiers sont modifiés
- Quels changements seront appliqués
- L'impact exact sur vos datapacks


![Modale d'Export](/images/patchnote/studio/2.1.0/view.webp)

### Quatre Modes de Vue

| Vue | Description |
| ---- | ----------- |
| **Différences** | Voir les changements ligne par ligne comparés au fichier original |
| **Vues de Débogage** | Deux pages de débogage pour l'inspection technique |
| **Éditeur de Code** | Un éditeur de code intégré expérimental, construit de zéro sans dépendances externes |

-# L'éditeur de code est expérimental. Il permet l'édition directe de fichiers dans Voxel Studio.

# Performance & Corrections de Bugs

### Optimisations React
Un effort significatif a été fait pour éliminer les re-renders inutiles dans toute l'application. De nombreux composants qui se re-rendaient à chaque interaction sont maintenant correctement optimisés, résultant en des performances drastiquement améliorées.

### Corrections de Bugs
- Correction des liens Discord pointant maintenant vers le bon serveur (désolé pour ça !)
- Nouveaux squelettes de chargement basés sur le design mis à jour
- De nombreux bugs de navigation et d'interface ont été corrigés
