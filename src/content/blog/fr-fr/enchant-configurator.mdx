# Libérez Votre Créativité avec le Configurateur d'Enchantements
Le **Configurateur d'Enchantements** est un outil qui vous permet de modifier et personnaliser les enchantements dans n'importe quel pack de données Minecraft et pas seulement ceux de l'écosystème Voxel.   
Que vous utilisiez un pack tiers populaire ou une création personnalisée, cet outil vous donne le plein contrôle sur vos enchantements et vous permet de les adapter à votre style de jeu unique.

### Conçu pour Tous pour une personnalisation complète
Avec le Configurateur d'Enchantements, vous pouvez entièrement personnaliser les enchantements d'un pack. 
- Ajustez les effets
- Modifier le niveau maximum des enchantements
- Changer l'obtention des enchantements
- Indiquer les items sur lesquels les enchantements peuvent être appliqués

Vous l'aurez compris, il a été conçu pour le grand public, l'outil vous offre la flexibilité de modifier les enchantements de manière simple, intuitive, et minimaliste, autrefois réservée aux développeurs expérimentés.  

### Toujours à Jour avec les Dernières Fonctionnalités
Nous nous engageons à améliorer continuellement le Configurateur d'Enchantements avec d'avantage de fonctionnalité, et des mises à jour régulières garantissent que vous disposez toujours d'un outil compatible pour les dernière version de Minecraft.   
Pour suivre toutes les mises à jour, consultez notre [changelog](https://voxel.hadel.io/fr-fr/update/enchant-configurator) et découvrez les nouveautés.
