# Documentation Interactive pour Yggdrasil
Nous sommes ravis de vous présenter notre nouvelle documentation interactive pour Yggdrasil, conçue pour rendre votre expérience de jeu plus transparente et intuitive.

## Exploration des Structures
La nouvelle documentation vous permet de **Visualiser chaque structure en détail** et de **Consulter les loots disponibles bloc par bloc**.

## Système de Loot par Bloc
Notre nouveau système de documentation permet une exploration précise :
- Cliquez sur n'importe quel bloc de récompense (Shulker, Coffre, Trial Spawner, Vault, Pottery) pour voir son contenu
- Visualisez les probabilités d'obtention de chaque item **(Soon)**
- Découvrez les items rares et leurs conditions d'apparition
- Explorez les différentes variantes de loot selon la difficulté
