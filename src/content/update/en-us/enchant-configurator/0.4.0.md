### New Feature in Version 0.4.0
The configurator has been deeply rewritten. You might not see any difference, but here's what it unlocks:
- Add new features in seconds or minutes.
- Have a specific configurator for each version of Minecraft.
- Create configurators for other aspects of the game in just a few hours.
- Switch instantly from one configurator to another on the same web page.
- In the future, a **configurator creator** will allow **anyone** to create their own configurator in seconds for their datapacks.

#### Added Features
- Added the ability to disable enchantments.
- Added a debugging menu.
- The slot section works perfectly.

#### Bug Fixes
- Fixed numerous bugs.
