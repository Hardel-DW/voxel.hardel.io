## Features for the Version 0.5.0

Part 2 of the configurator, the rewrite of enchant's configurator is now
complete.

- The configurator is now faster and lighter.
- The configurator is now data-file driven.

#### New Features

- The user interface is reorganized.
- The configurator is now data-file driven, it's execting for the future.
- The addon system is completely rewritten.
- The download button has been re-moved, it's now in the left sidebar.
- The debug button is now more discreet and also located in the sidebar.

#### Bug Fixes

- The "effects" has a bug, sometimes it disappeared when you disabled it.
- Some visuals didn't behave as expected.
- The "slots" tab had a bug in a specific case.
