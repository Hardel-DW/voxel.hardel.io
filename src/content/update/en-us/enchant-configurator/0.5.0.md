## New Feature in Version 0.5.0

The configurator continues to be rewritten; this is the second part of the
rewrite. Only a tiny part remains to handle multi-version support.

- The configurator is now faster and lighter.
- The configurator is now internally just a simple JSON file.

#### Added Features

- Changed the layout of the user interface elements.
- Donations are now possible.
- Addons have been completely rewritten.
- The download button has been moved to the bottom left of the page.
- The debug button is now less visible and more discreet.

#### Bug Fixes

- Effects disappeared when you disabled them.
- Visuals did not behave correctly.
- The slots tab had a bug in a specific case.
