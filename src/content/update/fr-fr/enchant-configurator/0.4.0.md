## Fonctionnalités de la version 0.4.0

Le configurateur d'enchants a été réécrit. Vous verrez peut-être pas la
différence, mais voila ce qui débarque :

- Pour l'équipe de développement, ajouter des fonctionnalités, nous prendra
  moins de temps.
- Il seras possible de créer un configurateur pour chaque version de Minecraft.
- Il seras aussi possible de créer de nouveau configurateurs pour d'autres
  aspects du jeu.
- Il seras possible de passer d'un configurateur à un autre sans quitter la
  page.
- À dans un futur proche, un système de **création de configurateurs** qui
  permettra à **n'importe qui** de créer son configurateur.

#### Nouvelles fonctionnalités

- Il est maintenant possible de désactiver les enchantements.
- Un nouveau menu de débogage a été ajouté.
- La section des emplacements a été corrigée.
