## Nouvelles fonctionnalités de la version 1.0.0

Le configurateur d'enchants est maintenant considéré comme stable, peu de bugs
devraient apparaître.

#### Nouvelles fonctionnalités

- L'intégralités des actions effectués sont maintenant stockés dans le contenu
  téléchargable. Ce qui vous permet de migrer votre configuration d'une version
  du pack a une autre.
- Les données sont aussi envonyés sur le servuer distant pour permettre de faire
  de l'équilibrage sur le contenu principal.
- Rajout d'un bouton "paramètres" dans la barre latérale, qui permet de modifier
  le nom du datapack/mods, ainsi qu'un bouton pour "Minifier" le code (Une
  pratique qui vide a supprimer les espaces, les commentaires, et autres
  éléments inutiles ce qui allège le contenu).
- Rajout d'un onglet "Non-Combinable" qui vous permet de gérer les enchantements
  qui ne peuvent pas être combinés ensemble.
- Rajout d'une section "Donation" pour permettre de rejoindre Patreon.
- Modifications des texts, pour être plus clair, et donner d'avantage
  d'informations.
- Le composant du "Compteur" permet maintenant de spécifier la valeur de votre
  choix comme un chant de texte.

#### Fonctionnalités Techniques

- Le systéme qui géres la traduction Anglais et Français est maintenant gérés
  par l'engine du configurateur, précédement Astro envoyés les traductions par
  le DOM a React. Et c'était pas optimale pour la bande passante, et pour le
  développement.
- La pages d'accueil s'affiche instantanément.
- Le "contexte" et globalement tout ce qui est lié a React est maintenant
  supprimé de l'engine du configurateur, ce qui permet de déporter le code vers
  une dépendence externe vu que c'est intégralement écrit en TypeScript pur.
- Les contextes react ont était déplacés vers un "store" zustand.
- Les actiosn/Condition sont maintenant en lowercase.
- Implémentation des tests unitaires pour chaque aspect du configurateur.
- Le code est aussi plus léger, et plus facilement maintenable.
