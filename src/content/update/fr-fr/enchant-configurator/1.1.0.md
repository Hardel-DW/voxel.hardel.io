## Nouvelles fonctionnalités de la version 1.1.0

Quelques nouvelles fonctionnalités ont été ajoutés, et certains bugs sont fixés.

#### Nouvelles fonctionnalités

- Rajout d'un nouveau boutons nommé "Mode" dans le menu principale, il permet
  definir l'obtention des enchants.
  - **Par défaut** : Les enchantments sont obtenu comme il a était prévu par le
    créateur, ou selon la configuration que vous avez faite.
  - **Supprimer** : L'enchantement n'a pas d'effets et il n'est plus obtensible.
  - **Creatifs** : Cette enchantment ne seras obtenable que par le menu
    créatifs.

- Dans le menu de débogages, un effets visuel indique si un élement est supprimé
  (Rouge), nouveau (Vert), modifier ou inchangé (Gris) par une couleur.

#### Correction de bugs

- Le systémes va maintenant correctement supprimer les fichiers, si il est vide.

### Technique :

- Refonte du systéme de traduction, il est intégrés directement dans le moteur,
  avant il était liés au Site. Et le site transférait les traductions au moteur.
  Par conséquent toutes les pages web sont alléger.
- Le Parser récupére la listes de tout les identifiants.
- Le Compiler, détermine pour chaque fichier si il est supprimé, nouveau,
  modifier, ou inchangé.
- Rajout d'un nouveau composant "Tabs".
- Rajout d'une nouvelle action "Alternative" qui permet de lancer des sous
  action selon l'égalités d'une variable.
- Certains morceau de code ont était réecrits.
