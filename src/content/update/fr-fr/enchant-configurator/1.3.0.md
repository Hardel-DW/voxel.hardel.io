# Version 1.3.0 - Migrations vers Breeze

## Breeze
Breeze est un une dépendence de Voxel, toutes la logique du moteur de Voxel a était déplacés vers cette dépendence, grâce a cela, nous pouvons maintenant utiliser le moteur de Voxel dans d'autres projets. Le code source de Breeze est disponible sur le GitHub.
Concrétement rien ne change pour vous, c'est une mise à jour interne de Voxel.

## Changement de l'interface
- Certains componsant sont maintenant plus détaillés et s'offre un nouveau look, c'est léger mais c'est les détails qui font toute la différence :3
- L'interface est maintenant plus responsive, elle s'adapte à toutes les tailles d'écrean
- Le barre latérale est maintenant plus petite pour les petits écrans
- Les compoonsants Shadcn/UI sont supprimer pour une solutions maisons
